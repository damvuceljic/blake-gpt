# Deck Proofing Issue Types

1. `Mismatch`
   - Commentary sign does not align with numeric representation.
2. `Outdated`
   - Narrative appears copied forward without current-month update.
3. `Missing`
   - Missing title, notes, disclaimer, or incomplete slide content.
4. `Consistency`
   - Definition/period/GAAP-IFRS labeling inconsistency.
5. `Clarity`
   - Content is present but ambiguous for executive interpretation.
