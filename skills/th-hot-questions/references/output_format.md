# Executive Brief Output Format

Use this exact shape by default:

1. Key insights (3-5 bullets)
2. Compact table:
   - Metric
   - Observation
   - Implication
3. Risks (max 3)
4. Opportunities (max 3)
5. Actions (max 3)
6. Likely hot questions + prepared answers (2-4 pairs)
7. End prompt:
   - `Is there any specific questions you'd like help coming up with an answer for?`

If confidence is medium/low:
1. Provide provisional interpretation.
2. Ask one clarifying question.
