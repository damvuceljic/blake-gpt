# Tokenization Guidelines

1. Preserve primary evidence while removing repetitive boilerplate.
2. Keep slide chunks concise:
   - title
   - key body text
   - note summary
3. Keep workbook chunks concise:
   - sheet summary
   - formula density
   - short formula preview
4. Store full normalized artifacts for traceability:
   - values CSV
   - formula CSV
   - chart series JSON
5. Never use raw binaries as default LLM input.
