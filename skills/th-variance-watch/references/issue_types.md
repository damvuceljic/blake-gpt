# Variance Watch Issue Types

1. `Mismatch`
   - Driver and bridge math does not reconcile.
2. `Outdated`
   - Variance assumptions likely rolled forward.
3. `Missing`
   - Commentary, schedule, or required values missing.
4. `Consistency`
   - Period basis or FX treatment is inconsistent.
5. `Clarity`
   - Driver logic is present but insufficiently explicit.
